git cheat sheet

learn more about git the simple way at rogerdudler.github.com/git-guide/

cheat sheet created by Nina Jaeschke of ninagrafik.com

create & clone

create new repository

clone local repository

clone remoterepository

git init

git clone /path/to/repository

git clone username@host:/path/to/repository

add & remove

add changes to INDEX

add all changes to INDEX

remove/delete

git add <filename>

git add *

git rm filename

commit & synchronize

commit changes

push changes to remote repository

connect local repository to remote repository

update local repository with remote changes

git commit -m "Commit message"

git push origin master

git remote add origin <sever>

git pull

branches

create new branch

obsi

switch to master branch

delete branch

push branch to remote repository

git checkout -b <branch>

e.g. git checkout -b feature_x

git checkout master

git branch -d <branch>

git push origin <branch>

merge

merge changes from another branch

view changes between two branches

obsi

git merge <branch>

git diff <source_branch> <target_branch>

e.g. git diff feature_x feature_y

tagging

create tag

obsi

get commit IDs

git tag <tag> <commit ID>

e.g. git tag 1.0.0 1b2e1d63ff

git log

restore

replace working copy with latest from HEAD

git checkout --<filename>
